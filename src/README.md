# Calculadora

## Instalación

### 1. Clona el repositorio:
 
   git clone https://github.com/AlbertoSueiro/Calculadora
   
### 2. Ve al directorio del proyecto:

  cd calculadora

### 3. Instala las dependencias:

  npm install

### 4. Inicia la aplicación:

  npm start
